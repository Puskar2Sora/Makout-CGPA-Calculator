# 🎓 MAKAUT Academic Grade System

A modern, elegant, and accurate **SGPA / CGPA Calculator** built specifically for **MAKAUT (Maulana Abul Kalam Azad University of Technology)** students.

Designed with a clean academic interface, this tool allows students to calculate semester results with full flexibility—supporting **different subjects every semester**, labs, credit-based grading, and **official-style PDF grade cards**.

> ⚡ Lightweight • 📄 Accurate • 🎯 MAKAUT-compliant • 💻 Fully client-side

---

## 🌟 Why This Project?

Most online CGPA calculators:
- Assume fixed subjects
- Ignore lab credits
- Don’t match MAKAUT logic
- Look outdated or cluttered

This project solves all of that by offering:

✔ Flexible subject entry (theory + lab)  
✔ Credit-based calculations  
✔ Semester-wise isolation  
✔ Clean academic UI  
✔ Downloadable grade card PDF  

Built **by a MAKAUT student, for MAKAUT students**.

---

## ✨ Key Features

### 📘 Academic Accuracy
- MAKAUT-compliant **grade-to-point mapping**
- Credit-index based **SGPA calculation**
- Percentage conversion using MAKAUT standard
- CGPA displayed alongside SGPA

### 🧠 Smart Semester Handling
- Supports **Sem 1 to Sem 8**
- Each semester maintains **independent subject data**
- Switch semesters without losing input
- Unlimited subjects & labs per semester

### 🧾 Grade Card PDF
- One-click **Download Grade Card**
- Includes:
  - Subject list
  - Credits
  - Grades
  - Total credits
  - Credit points
  - SGPA, Percentage, CGPA
- Clean, printable, academic layout

### 🎨 UI & UX
- Professional university-style interface
- Clean tables & spacing
- Minimal distractions
- Fully responsive (desktop-friendly)
